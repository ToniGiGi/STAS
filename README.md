**STAS (Strategic Talent Auto-Selector)** es una plataforma de software integral diseñada específicamente para optimizar los procesos de Recursos Humanos en la **industria automotriz (Tier 1 y OEM)**. 

Este repositorio contiene la **Plataforma Web Administrativa**, la cual sirve como el panel de control central (Panel de RH) dentro del ecosistema STAS. (Nota: Los reclutadores en campo y los candidatos interactúan a través de la App Móvil complementaria).

## ✨ Características Principales

* 📊 **Panel de Control (Dashboard):** Analíticas en tiempo real del flujo de talento, compatibilidad y estatus de vacantes.
* 💼 **Gestión de Vacantes:** Creación de perfiles de puesto técnicos altamente detallados (Core Tools, IATF 16949, etc.) con cruce de compatibilidad automatizada.
* 👥 **Directorio de Talentos (3FN):** Base de datos relacional robusta con filtros dinámicos por departamento, experiencia y certificaciones técnicas.
* 🧠 **Catálogo de Soft Skills:** Módulo de evaluación de habilidades blandas (Liderazgo, Resolución de Conflictos, Inteligencia Emocional) adaptado a entornos de manufactura de alta presión.
* ⚙️ **Algoritmo Parametrizable:** Motor de ponderación personalizable donde RH calibra el peso porcentual de la experiencia, técnica y cultura.
* 📱 **Conexión Multicanal:** Generación de credenciales y accesos para el ecosistema móvil.

## 🛠️ Stack Tecnológico

* **Backend:** Laravel (PHP)
* **Base de Datos:** MySQL (Arquitectura Normalizada 3FN)
* **Frontend:** HTML5, CSS3 (Diseño Custom "Enterprise"), JavaScript (Vanilla, Fetch API)
* **Librerías Adicionales:** * [Chart.js](https://www.chartjs.org/) (Gráficas de Radar para inspección de perfiles)
    * [SweetAlert2](https://sweetalert2.github.io/) (Modales y alertas interactivas)
    * FontAwesome (Iconografía)
